ua_parser C++ Library
=====================

Usage
-----

To build the (static) library:

    $ make uaparser_cpp

To build and run the tests:

    $ make test

Dependencies
------------

* yaml-cpp (0.5 API)
* glog (for the `CHECK` macro)
* gtest (for testing)
